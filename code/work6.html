ポートフォリオ

web

2023.08

担当範囲

デザイン/コーディング

制作期間

2ヶ月

考えや想いなど頭や心の中にある、目に見えず形のないぼんやりとしたものにキャンバスのうえで色をつけていくイメージでポートフォリオを作成しました。言語はHTML、CSS、Javascriptを使用しています。水彩画で描いているような柔らかい色味を意識し、一度描いた上からもその時々に応じて上から色を重ねてより良いものに仕上げ長く愛せるようなデザインになるよう想いを込めています。また、色をつけるというイメージから、色づく前はグレースケールにしていますが、ただのグレーではなく温かみを感じさせるよう少し赤みがかったグレーで統一しています。
